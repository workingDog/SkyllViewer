#  SkyllViewer


A demo of the [SwiftSkyllKit](https://github.com/workingDog/SwiftSkyllKit) library in a SwiftUI app searching for AI agents SKILL.

[SwiftSkyllKit](https://github.com/workingDog/SwiftSkyllKit) is a lightweight Swift package for searching and retrieving AI agent skills.
It is a Swift API interface to the skill discovery platform for AI agents [Skyll](https://www.skyll.app/).
